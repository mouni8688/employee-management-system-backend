# 🚀 Employee Management System (Backend)

A role-based Employee Management System built using Node.js, Express, and MongoDB.  
This system allows admins to assign tasks and monitor employees, while employees can manage their tasks and provide feedback.

---

## 📌 Features

- 🔐 JWT Authentication (Login/Register)
- 👥 Role-based Access (Admin / Employee)
- 📋 Task Management System
  - Admin can assign tasks
  - Employee can view assigned tasks
  - Employee can update task status
- 💬 Feedback System
  - Employees submit feedback
  - Admin can view all feedback

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JSON Web Tokens (JWT)
